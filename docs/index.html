gh-sks

GitHub SSH Key Sync for Linux Servers

Automatically sync GitHub users' public SSH keys into your server's ~/.ssh/authorized_keys. Runs automatically hourly. 

curl -fsSL https://raw.githubusercontent.com/BenDutton/gh-sks/main/install.sh | sudo bash
Full Documentation View on GitHub
1
Read mappings

Reads user mappings from /etc/gh-sks/github_authorized_users.

2
Fetch keys

Fetches each GitHub user's public keys from github.com/<user>.keys.

3
Merge safely

Merges keys into a managed block in ~/.ssh/authorized_keys, preserving any manually-added keys.

Configuration
# /etc/gh-sks/github_authorized_users
azureuser octocat
azureuser defunkt
deploy    torvalds
Update
sudo gh-sks --update
Uninstall
sudo gh-sks --uninstall

Requires bash 4.0+, curl, systemd, and network access to github.com.
